# 📂 Python File Handling CRUD Project

A simple command-line **File Management System** built with Python that demonstrates the fundamentals of file handling. The application allows users to create, read, update, rename, append, overwrite, and delete text files through an interactive menu.

This project was built to strengthen my understanding of Python file operations, exception handling, user input, and the `pathlib` module.

---

##  Features

-  Create new text files
-  Read existing files
-  Update file contents
  - Rename a file
  - Overwrite file contents
  - Append new content
-  Delete files
-  Display all files and folders in the project directory
-  Exception handling for invalid operations
